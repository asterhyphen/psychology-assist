# Calmora
Built with clean architechure, vibecoded and improved in 24h as part of Xypheria Hackathon at Alvas. A special theme similar to the Xypheria Hackathon is also made under the name "Journal". </br>

## Architecture
- **State Management**: Riverpod
- **Navigation**: Bottom tab bar with smooth transitions
- **Database**: SQLite (prepared, ready for mood history)
- **Notifications**: Local notifications for check-ins & reminders
- **AI Model**: Ollama (LLAMA 3.0)


## 🔧 Tech Stack

- **Framework**: Flutter 
- **State Management**: Riverpod 
- **Database**: SQLite 
- **AI Model**: Ollama (Llama 3.2)
- **Notifications**: flutter_local_notifications
